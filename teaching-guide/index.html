第一轮 · 生成初版
完整需求描述 → 获得可运行的 HTML 应用
把受众、功能、界面、技术要求一次说清楚
▾
受众：3-8岁 自闭症谱系 Web Audio API 单 HTML 文件 横屏布局
提示词
请为自闭症谱系儿童制作一个五声音阶互动 HTML 应用，要求： 1. 受众：自闭症谱系儿童（3-8岁），注意力短、需要强视觉引导 2. 音阶体系：宫商角徵羽（do re mi sol la），对应五种颜色 3. 交互流程分三步递进： - 第一步「逐音认知」：展示每个音对应的卡通形象动作， 孩子自由点击任意音，点哪个就展示哪个 - 第二步「听音模仿」：系统播放一个音，给出动作提示， 孩子凭提示找到对应按钮 - 第三步「自由按奏」：孩子随意按，系统记录并可回放完整旋律 4. 界面要求：横屏布局，左边大面积展示引导图，右边彩色大按钮 5. 视觉风格：色彩鲜明、圆角、大按钮（适合触屏操作）、有进度提示 6. 音频：使用 Web Audio API 生成对应频率的三角波音色 7. 输出：单个 HTML 文件，可直接在浏览器打开
💡 把受众特点写进去（注意力短、触屏操作）能让 AI 自动调整界面风格
⚡ 要求「单 HTML 文件」是关键——直接双击打开，无需配置服务器
R2
第二轮 · 加入柯达伊手势
卡通熊 + 五种手势图片 → 增强视觉引导
先生成 5 张 AI 图片，再嵌入应用
▾
柯达伊手势 卡通熊形象 AI 图片生成 图片嵌入
提示词
第一步左边放一只大卡通熊形象，做柯达伊手势动作： - do：握拳 - re：双手放胸前，手腕向上翘，手指并拢朝上，手心朝前 - mi：三指张开 - sol：手臂打开，五指并拢，手心朝外 - la：五指完全张开 用 AI 生成 5 张不同手势的卡通熊图片，嵌入应用中， 点哪个音就切换到哪个熊的图片。
🎨 图片生成和代码修改可以分两步做：先用 AI 生图，再告诉 AI 把图片路径替换进去
💡 柯达伊手势是国际通用的音乐教学法，每个音有标准手势——孩子通过身体记忆音高

▾
交互修正 自由探索
提示词
第一步改成自由探索模式： 按哪个音就展示那个音的熊，不要自动跳到下一个。
🔑 修正提示词要「描述行为」不是「说感受」：说「按哪个音就展示哪个」比「感觉怪怪的」精准得多
⚡ 短而精准的修改指令效果往往比长段描述更好
R4
第四轮 · 图片手势修正
Re 和 Sol 手势重新生成 → 方向细节校准
图片不对就重新描述，AI 重新生图替换
▾
图片重生成 手势方向 Re / Sol
Re 的修正提示词
Re 的熊：双手放胸前，手腕翘起，手指并拢朝上， 手心朝前，两只手方向一致都向上。
Sol 的修正提示词
Sol 的熊：手臂打开向两侧伸展，五指伸直并拢不分开，手心朝外。
🔍 AI 对方向性描述（朝上/朝前/朝外）很敏感，多加一个细节往往就能得到正确结果
🔁 图片修正可能需要 2-3 次迭代，每次只修改不对的地方，其他保持不变
